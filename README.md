```bash
# Clone the repository
git clone https://github.com/FairTradeWorker/fairtradeworker.git
cd fairtradeworker

npm install

# Copy environment variables
cp .env.example .env

# Add your API keys to .env

npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Environment Variables

Create a `.env` file in the root directory:

```bash
# Trueway Routing API (get from api.market)
VITE_TRUEWAY_API_KEY=your_trueway_api_key

# Stripe (for payment processing - planned)
VITE_STRIPE_PUBLIC_KEY=your_stripe_public_key

# Optional: Sentry for error tracking
VITE_SENTRY_DSN=your_sentry_dsn
```

### API Keys

1. **Trueway Routing API**
   - Visit [API.market](https://api.market/)
   - Search for "Trueway Routing API"
   - Subscribe to a plan (free tier available)
   - Copy your API key to `.env`

2. **Stripe** (when implementing payments)
   - Create account at [Stripe](https://stripe.com)
   - Get your publishable key from dashboard
   - Add to `.env` file
